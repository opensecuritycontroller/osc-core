0) Deploy ovf as regular VM

----------------------------------------

1) Set temporary network info:

ifconfig eth0 10.71.86.149 netmask 255.255.255.0
route add default gw 10.71.86.252 eth0
ping 10.71.86.252
----------------------------------------

2) Remove password:

/usr/local/bin/updateconfig.sh /etc/passwd
vi /mnt/config/etc/passwd

Add:
#nobrk1n:x:0:0:Debug User:/root:/usr/local/bin/clidebug
nobrk1n:x:0:0:Debug User:/root:/bin/bash

--------------------------------------------------

3) Modify rc.local startup script

/usr/local/bin/updateconfig.sh /etc/rc.d/rc.local
vi /mnt/config/etc/rc.d/rc.local

Add:
echo 'Starting vmiDC agent'
cd /mnt/config/opt/vmidc/bin
chmod +x ../jre/bin/java
../jre/bin/java -Djava.security.egd=file:/dev/./urandom -jar vmiDCAgent.jar > /dev/null 2>&1 &

--------------------------------------------------
4) Modify vi.local

/usr/local/bin/updateconfig.sh /usr/local/bin/iv.local
vi /mnt/config/usr/local/bin/iv.local

Add:
/usr/local/sbin/iptables -I INPUT -i eth0 -p tcp --dport 8090 -j ACCEPT

--------------------------------------------------

5) Copy over opt/* to /mnt/config/opt/*
6) Delete generated files (log, tomcat-work, server.pid, etc). Tip: User BeyondCompare.
7) Shutdown VM.
8) Export to OVF.
9) Copy *.vmdk file ONLY to vmiDCServer\ovf directory.
